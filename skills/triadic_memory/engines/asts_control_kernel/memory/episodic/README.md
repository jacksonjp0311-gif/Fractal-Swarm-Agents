# Episodic Memory
This directory contains placeholder episodic memory primitives.

## What it does
- Defines interfaces for storing, retrieving, replaying, and consolidating episodes.

## Mini directory
- `store.py`
- `retrieval.py`
- `replay.py`
- `consolidate.py`
- `episode.py`
- `boundaries.py`

## Notes
